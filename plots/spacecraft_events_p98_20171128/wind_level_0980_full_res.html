Finding χ2 Minimization



The program for finding the χ2 minimum for events between spacecraft works by using a primary spacecraft to find events. I selected Wind because I used Wind to train the model, and the events found in Wind are consistent with events found by eye or independent spacecraft algorithms. I define an event discontinuity as a time where our model indicates a 98.00% probability the solar wind parameters corresponds to a 5σ discontinuity in a component of the magnetic field. Currently, I find 55 events using the Wind spacecraft to define an event. Then I create a grid of solar wind speed samples for ± 40 minutes around a event. However, if the spacecraft does not have reliable flow speed measurements I use the magnitude of the magnetic field. Then I shift the other 3 spacecraft onto the Wind using the course grid. Then I select a 300 minute (±90 minutes) window around the shifted Wind and other spacecraft and derive a first order χ2 minimum. Next, I use the first order χ2 minimum and the same parameter (i.e. preferably flow speed) and sample ± 20 minutes around the first minimum with a 150 minute window. The new χ2 minimum is the second order minimum. Using the second order χ2 minimum, I create a fine grid 30 minutes (± 15 minutes) around the rough χ2 minimum time for DSCOVR and ACE, but a 50 minute window (±25 minutes) for SOHO/CELIAS because SOHO/CELIAS does not take magnetic field observations. The fine grid cadence is set to the maximum sampling frequency of a given spacecraft. Then the program stores the χ2 and cuts the grid and window by half twice more (i.e. 3 refined χ2 minimizations. Globally if the first or last point is the χ2 min. then the program expands the time search window by 20 minutes. The χ2 reported is the minimum for the last refined time.

Table Summary

Each table contains a row for each spacecraft with observations in the requested time period. Then each row contains 6 columns. The first column labels the spacecraft for a given row. Then next column (Obs. Time [UT]) gives the time of the "event" in the reference frame of the spacecraft. Offset is the offset in minutes from the Wind observation. P-val (plasma) is the p-val for a 5σ discontinuity at the reference Obs. time for a given spacecraft's measured plasma values. P-val (mag.) is the p-val for a 5σ discontinuity given at the reference Obs. time for a given spacecraft's measured magnetic field values. Finally, an X in Use Plasma Mod. means the minimization routine use the wind Speed to find the best match χ2 time, while an empty cell denotes using the magnetic field for rough characterization. The final 6 parameters are the absolute maximum values of the discontinuities for ±90 seconds around an event. They are a rough guide to what the plasma and magnetic field did during a particular event. However, clicking the Event on YYY/MM/DD HH:MM:SS shows the plasma and magnetic field for two different time windows (± 3 hours and ± 25 minutes). Finally, clicking on the spacecraft name shows the χ2 minimization as a function of time for the wide and refined windows.
